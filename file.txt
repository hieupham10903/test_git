Tóm tắt các lệnh:
Tạo SSH key (nếu chưa có):

bash
Copy code
ssh-keygen -t ed25519 -C "your_email@example.com"
cat ~/.ssh/id_ed25519.pub
Thêm SSH key vào GitHub.

Clone repository bằng SSH:

bash
Copy code
git clone git@github.com:username/repository-name.git
cd repository-name
Thêm và đẩy file lên GitHub:

bash
Copy code
echo "Nội dung của file" > example.txt
git add example.txt
git commit -m "Thêm file example.txt"
git push origin main
